# Cybersecurity Internship Projects  
**Arch Technology Cybersecurity Internship**

## Overview
This repository contains two educational cybersecurity projects developed for the Arch Technology Cybersecurity Internship:

1. **Task 1 – Basic Network Sniffer**  
   A Python project using Scapy to capture and analyze network packets. Logs include source/destination IP, protocol, and ports.

2. **Task 2 – Basic Keylogger**  
   A Python project using pynput to simulate a keylogger in a controlled environment. Logs keystrokes with timestamps for educational purposes.

---

## Folder Structure

- `task1-network-sniffer/` – Network sniffer project  
- `task2-keylogger/` – Keylogger project
